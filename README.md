# SyncWatch 🎬

O **SyncWatch** é uma plataforma inteligente de curadoria e rastreamento de filmes e séries, desenhada para conectar amigos e otimizar o tempo gasto decidindo "o que assistir hoje". Mais do que um simples tracker, o SyncWatch utiliza análise de preferências individuais para gerar recomendações cruzadas, transformando a experiência de streaming em um evento social.

## 🚀 O Problema

Quantas vezes você e seus amigos perderam mais tempo escolhendo um filme do que realmente assistindo a ele? O SyncWatch resolve a indecisão através de algoritmos de afinidade, comparando gostos individuais para sugerir conteúdos que agradem a todos os membros do grupo.

## 🛠️ Tecnologias Principais

- **Frontend:** Next.js (App Router), Tailwind CSS, Shadcn/UI.
- **Backend & Auth:** Supabase (PostgreSQL, Row Level Security).
- **Dados:** TMDB API (The Movie Database).
- **Infraestrutura:** Vercel (CI/CD Automático).
- **Cache & State:** TanStack Query.

## ✨ Funcionalidades (MVP)

- **Onboarding Inteligente:** Questionário de preferências integrado à API do TMDB.
- **Rastreamento Pessoal:** Listas customizáveis (Para assistir, Assistindo, Finalizado).
- **Dashboard de Progresso:** Acompanhamento de temporadas e episódios.
- **Listas Compartilhadas:** Grupos para amigos sincronizarem seus status de visualização.
